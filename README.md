# Sugarbaby's Childcare

A modern, responsive Single Page Application (SPA) built to serve as the digital storefront for a local childcare center in Waterloo, IA.

Designed with busy parents in mind, this application provides a seamless, mobile-first user experience, allowing users to easily view facility programs, read admission policies, and submit inquiries directly to the business owner.
